## 🎭 السيناريوهات المغطاة

### SQL Injection:
1. **Error-Based**: استخراج معلومات من الأخطاء
2. **Union-Based**: استخراج بيانات من جداول أخرى
3. **Boolean-Based Blind**: استخراج البيانات عبر الفروق في الرد
4. **Time-Based Blind**: استخراج البيانات عبر التأخير الزمني
5. **Out-of-Band**: إرسال البيانات إلى خادم خارجي

### XXE Injection:
1. **قراءة ملفات النظام**: الوصول لملفات النظام
2. **SSRF**: إجراء طلبات شبكة داخلية
3. **XML Bomb**: هجمات DoS
4. **Out-of-Band**: استخراج البيانات عبر قنوات خارجية
