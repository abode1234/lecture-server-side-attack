SQL Injection + XXE Injection

محاضرة شاملة عن حقن SQL وثغرات XXE - ساعتان

مقدمة عن الثغرات الأساسية

مرحباً بكم في المحاضرة الأولى من سلسلة أمان تطبيقات الويب. في هذه المحاضرة، سنتعلم عن نوعين من أخطر الثغرات:

SQL Injection

ثغرة تسمح للمهاجم بحقن أوامر SQL خبيثة في استعلامات قاعدة البيانات.

XXE Injection

ثغرة في معالجة XML تسمح بالوصول لملفات النظام وتنفيذ هجمات SSRF.

أساسيات SQL Injection

خطورة SQL Injection

SQL Injection يمكن أن يؤدي إلى تسريب بيانات كاملة، تدمير قواعد البيانات، أو السيطرة على الخادم.

# مثال على SQL Injection أساسي
' OR '1'='1' --
' UNION SELECT username, password FROM users --
'; DROP TABLE users; --

أساسيات XXE Injection

خطورة XXE

XXE يمكن أن يؤدي إلى قراءة ملفات النظام، SSRF attacks، أو DoS attacks.

<!-- مثال على XXE payload -->
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE foo [ <!ENTITY xxe SYSTEM "file:///etc/passwd"> ]>
<stockCheck><productId>&xxe;</productId></stockCheck>
